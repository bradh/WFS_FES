|*Requirement {counter:req-id}* |/req/core/fc-time-definition +

The operation SHALL support a parameter `datetime`
with the following characteristics (using an OpenAPI Specification 3.0 fragment):

[source,YAML]
----
name: datetime
in: query
required: false
schema:
  type: string
style: form
explode: false
----
